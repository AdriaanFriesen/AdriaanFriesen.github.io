Introduction

Python is an object oriented interpreted langauge, meaning it's not compiled like C or Java is, but just run as is. It it fairly easy to learn compared to other languages, and is a very powerful tool when raw execution speed is not the main concern.

Basic Syntax and Functions

First, lets start with the print() function. As you might expect, it prints whatever's inside to the console. To print text, the text to be printed must be inside of either single quotes or double quotes. Here's an example:

print("Hello World!")

Output:
Hello World!

It prints Hello World! to the console. We can also print variables to console. But how do we set a variable? Like this!

var_1 = "Hello"
var_2 = "World"
print(var_1)

Output:
Hello

Notice how there were no quotes when we used the name of a variable. You may have also noticed how there were two variables, and they form Hello World together, but how do we put them together? We can use the + operator on strings to concatenate them to each other.

var_1 = "Hello"
var_2 = "World"
print(var_1 + var_2)

Output:
HelloWorld

That works, but there was no space in between. All we have to do to fix this is add a space between them. We'll add a space into the print function.

var_1 = "Hello"
var_2 = "World"
print(var_1 + " " + var_2)

Output:
Hello World

Variables can be of other types as well. For example, we can set a variable to an integer.

number = 5
print(number)

Output:
5

We can also perform operations on numbers, like this:

number = 17
other_number = 5
new_number = number + other_number
print(new_number)

Output:
22

Now lets say that we want to print both letters and numbers. This might be your first guess:

word = "hello"
number = 4
print(word + number)

Output:
Traceback (most recent call last):
File "<string>", line 3, in <module>
TypeError: can only concatenate str (not "int") to str

It didn't work. But why? Python can only add varables of the same type. If we check the type of our varables using the type() function, we can see that they are different.

word = "hello"
number = 4
print(type(word))
print(type(number))

Output:
<class 'str'>
<class 'int'>

Now we see that word is of type str and number is of type int. Can we covert number to a str type so it can be printed? Yes!

word = "hello"
number = 4
print(word + str(number))

Output:
hello4

We used the str() function to convert the variable given to a str type so it could concatenate to word. There are other types of variables, like floating point numbers, which can contain decimals, lists, which host any number of other variables inside them, dictionaries, which contain a number of key-value pairs, and more.


In-depth Variables

In Python, there are many types of variables. We have already seen int and str types of variables, but there are others. Lets first take a look at a float type variable:

float_number = 4.76
print(float_number)
print(type(float_number))

Output:
4.76
<class 'float'>

We can see that we were able to store a floating point number and see that it was of type float. Lets take a look at the code below.

num_1 = 7
num_2 = 2
num_3 = num_1 / num_2
print(type(num_1))
print(type(num_2))
print(type(num_3))
print(num_3)

Output:
<class 'int'>
<class 'int'>
<class 'float'>
3.5

We can see that even though we started with two integers, we ended with a float. This is an example of Python's implicit variable declaration. Even though we didn't explicitly tell Python to make num_3 a floating point number, it did it anyways because 7 divided by 2 is 3.5, which is not an integer. Sometimes we dont want Python to use the most specific type of variable possible, for example putting an integer into a float type variable. We can use the variable functions we saw in the last chapter to do this.

int_in_a_float = float(4)
print(int_in_a_float)
print(type(int_in_a_float))

Output:
4.0
<class 'float'>

Notice how float variables always have decimal points, even when there is no decimal value. Lets now take a look at list type variables. They hold any number of other variables inside them, and can be changed after they are created. To make a list, we put brackets around the items of the list, which are separated by commas.

first_list = [5, 4.9, "youch"]
print(first_list)

Output:
[5, 4.9, 'youch']

Now lets say that we want to access a specific list item. We can use list indexes to do this. To use a list index, we put the list name and a number inside brackets at the end of the name where the number is the index of the list item we want to access. It's easier explained through an example:

first_list = [5, 4.9, "youch"]
print(first_list[1])

Output:
4.9

Python gave us the list element with index 1. This was not the first list element though, because indexes in Python start at 0. This means that youch has a list index of 2. We can even put lists inside of other lists to create nested lists. To change a list is to set a list index equal to a different value.

other_list = ["one", 2, 3]
print(other_list)

Output:
['one', 2, 3]

Now we have a list with an arguably incorrect element. Lets change this to be the "correct" value.

other_list = ["one", 2, 3]
print("Wrong list:")
print(other_list)
other_list[0] = 1
print("Correct list:")
print(other_list)

Output:
Wrong list:
['one', 2, 3]
Correct list:
[1, 2, 3]


Loops and Conditionals

Loops and conditional statements are an important part of any programming language, and Python is no different. First lets start with the if statement. It takes a variable or other conditional and performs the functions inside of it if the input is true. We'll also be introducing boolean variables, which store either True or False.

boolean_var = True
if boolean_var:
print("Hello")

Output:
Hello

Since boolean_var was True, the if statement executed what was inside it. Notice how in Python, loops and conditionals use whitespace to indicate what is and what is not inside the statement. This is why there was an indent before print("Hello").

if statements can also can code that executes if the check failed. This is the else keyword.

new_boolean = False
if new_boolean:
print("Hello")
else:
print("Goodbye")

Output:
Goodbye

The code inside the else statement executed because the if check failed. There is one more statement that goes with if, which is the elif keyword. It is executed if the initial check fails, and has its own check. There can be unlimited elif statements in a single if statement.

response = 2
if response = 1:
print("Good Morning")
elif response = 2:
print("Good Afternoon")
elif response = 3:
print("Good Night")

Output:
Good Afternoon

That's the if statement. Now lets see the while loop. It executes the code inside it until the conditional given to it is false. We'll also see the += operator. It adds whatever is on the right of the operator to what's on the left. As an example, var = var + 1 is equivalent to var += 1.

counter = 0
while counter < 5:
counter += 1
print(counter)

Output:
1
2
3
4
5

The final loop we'll look at is the for loop. It performs an action for every variable in an iterable. An iterable is just something that can be accessed through an index value, for example the lists we looked at in the last chapter.

numbers = [1, 2, 3, 4, 5]
total = 0
for number in numbers:
total += number
print("List:")
print(numbers)
print("List total:")
print(total)

Output:
List:
[1, 2, 3, 4, 5]
List total:
15

We successfully added the contents of the list into one number. This is one of many things for loops can do.


External Libraries

One of the core design philosophies of the Python language is that the base set of functions should be lightweight, and imports should be relied upon instead. This is what the import keyword is for. They go at the top of your file, and import all the functions and classes from the referenced library. There are multiple ways to obtain libraries, but that won't be covered in this tutorial. Instead, we'll use one of the many libraries that are included with a standard installation of Python. We'll be demonstrating imports with the random library. As you might guess, it does all sorts of things with random numbers.

import random

Now we've imported the random module. Lets do something with it, like the randint() function. It takes two integers as input, the minimum and maximum number it can generate. For example, randint(0, 5) would generate a random integer between 0 and 5, inclusive.

import random
random_number = random.randint(0, 7)
print(random_number)

Output:
2

We rolled a 2, but it could have been any number between 0 and 7. You might have also noticed that we used random.randint() instead of just randint(). This is how you tell Python that the function you're using comes from a library, by putting the library name, a period, then the function name.
